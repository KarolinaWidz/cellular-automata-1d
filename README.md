# Cellular automaton
Cellular automaton simulation using JavaFX. 
- Cellular automaton 1D
- Grain Growth 2D
- Smoothing edges using Monte Carlo method

App is available on: https://karolinawidz.github.io/cellular-automaton/

## Technologies
- Java 8
- JavaFX
- Maven
- TestFX
- Lombok (version 1.18.12)
